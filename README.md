   - 總覽說明

##  快速使用指南

### 情況一：已經點擊過 TRANSCEND.lnk（系統已感染）

1. 以系統管理員身分執行 `方案1_清理已感染系統.ps1`
2. 等待清理完成
3. 重新啟動電腦
4. 繼續執行「情況二」的步驟清理 USB

### 情況二：還沒點擊 TRANSCEND.lnk（系統未感染）

1. **不要點擊** USB 上的任何 `.lnk` 檔案！
2. 以系統管理員身分執行 `方案2_備份USB資料.ps1`
3. 確認備份完成後，格式化 USB
4. 將備份資料複製回 USB

##  注意事項

1. **所有腳本都需要以系統管理員身分執行**
   - 右鍵點選 `.ps1` 檔案
   - 選擇「以系統管理員身分執行」

2. **執行前可能需要調整 PowerShell 執行原則**
   
   如果出現「無法載入檔案，因為這個系統上已停用指令碼執行」錯誤：
   
   ```powershell
   # 以系統管理員身分開啟 PowerShell，執行：
   Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
   ```

3. **備份腳本中的磁碟機代號**
   
   `方案2_備份USB資料.ps1` 預設使用 `E:` 磁碟機
   
   如果你的 USB 是其他代號（例如 `F:`），請修改腳本第 6 行：
   ```powershell
   $USBDrive = "F:"  # 改成你的 USB 磁碟機代號
   ```

##  病毒資訊

### 病毒類型
捷徑病毒（LNK Virus / Shortcut Virus）

### 感染方式
1. 隱藏真實資料夾
2. 建立同名捷徑檔案 (.lnk)
3. 使用者點擊捷徑時執行病毒腳本
4. 病毒複製自身到系統目錄
5. 加入 Windows Defender 排除清單

### 危害
- 竊取系統資訊
- 感染其他 USB 裝置
- 下載額外惡意軟體
- 建立後門程式

##  疑難排解

### Q: 執行腳本時出現紅色錯誤訊息
A: 確認是否以「系統管理員身分」執行，並檢查 PowerShell 執行原則

### Q: 找不到病毒檔案
A: 可能尚未被感染，或病毒已被 Windows Defender 隔離

### Q: 備份後資料無法開啟
A: 檢查備份資料夾中的檔案屬性，可能需要移除「隱藏」或「系統」屬性：
```powershell
attrib -h -s "備份資料夾路徑\*.*" /s /d
```

### Q: 格式化後病毒又出現
A: 表示系統仍被感染，請先執行「方案1」清理系統
