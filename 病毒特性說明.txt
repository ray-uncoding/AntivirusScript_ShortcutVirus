中毒者會在點擊 .lnk 後被複製病毒並且該病毒有以下特性

1. 此病毒被安裝到系統後，將會自動加入到 windows defence 的忽略列表中
2. 任何在此資料夾下的檔案將被轉移至隱藏資料夾中
3. 複製自身至系統後，將會亂數改名，確保不被追蹤

結構：
1. /System Volume Information  # usb 系統原生檔案夾
2. /syscolume # 病毒偉造資料空間
    | - u105833.bin # 病毒的開機執行檔案
    | - u146141.dat # 病毒的常駐執行檔案
    | - u186269.vbs # 病毒的啟動
    | - u863864.bat # 病毒的啟動
3. TRANSCEND     # 原本的資料夾
4. TRANSCEND.lnk # 病毒的啟動，偽裝成連結檔案，會開啟項目 3

由於任何在這個資料夾寫的殺毒腳本都會被感染
